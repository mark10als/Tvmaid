Tvmaid インストールマニュアル

インストール

TVTestの設定

TVTestでTVを視聴できる状態にしてください。

注意
地域によっては、TVTestのチャンネルスキャンで同じ放送局が複数取得できることがあります。
Tvmaidはこれらを同時に使えません。1つを残して他は無効(チェックを外す)にしてください。

Tvmaidのセットアップがうまくいかない時は、TVTestのバージョンを確認してください。

必要なプログラムのダウンロードとインストール

Microsoft .NET Framework 4 (Web インストーラー)
http://www.microsoft.com/ja-jp/download/details.aspx?id=17851

「Microsoft .NET Framework 4 Client Profile」というのもありますが、それではダメです。


Visual Studio 2013 の Visual C++ 再頒布可能パッケージ
http://www.microsoft.com/ja-jp/download/details.aspx?id=40784

(vcredist_x86.exeをインストールしてください)

Tvmaidの配置

Tvmaidのアーカイブを適当なフォルダに解凍します。
Vista以降では、Program Filesフォルダには入れないでください。

XP以外(Vista、7、8)の場合、Tvmaidフォルダの「web.bat」を管理者で実行してください。
これは、Tvmaidの内蔵Webサーバを実行可能にする設定です。

セットアップ

Setup.exeを実行してください。

TVTestの場所と録画フォルダを指定します。

次にチューナを指定します。
Bonドライバを指定して、チューナ名を入力してください。
以下の点に注意してください。

Bonドライバ

同じBonドライバを複数指定できます。ただし、TVTestで同時に使用できる場合です。
Bonドライバの仕様によります。
例えば、BonDriver_PT-ST（人柱版３）は同時使用が可能ですが、古いBonドライバだと同時使用できないものもあります。
この場合は、リネームして使用してください。

Bonドライバのファイル名のみを指定できます。
「ファイル名のみを追加」にチェックを入れてください。
「TVTest検索フォルダ」にあるBonドライバのみ、この指定ができます。

チューナ名

チューナ名に「=」は使えません。
同じチューナ名を複数指定できません。

チューナの指定で、順番は意味があります。
録画予約は、上に指定したものから順番に使われます。

チューナの設定が終わったら、設定終了ボタンを押してください。
チューナ情報を読み込み、Tvmaidが起動します。
また、自動的にTvmaid用プラグインが、TVTestへコピーされます。

番組表取得

右クリック→番組表取得を選択し、しばらく待ちます。
これで使える状態になります。

チューナの状態を見るには、タスクトレイアイコンをダブルクリックし、チューナモニタを表示してください。
番組表は、http://localhost:20000/maid/epg.htmlをブラウザで開いてください。

その他の操作方法は、マニュアルを参照してください。

番組表取得で、チューナモニタにエラーが表示される場合は、チューナの設定が間違っていることが考えられます。
Setupを再度実行し、チューナの数、ドライバ名等を再確認してみてください。


Tvmaidは常時起動しておいてください。
再起動時、自動で起動するように、スタートアップに登録してください。

アンインストール

web-del.batを管理者で実行してください。
web.batで登録した設定を削除します。

TVTestのPluginフォルダのTvmaidPlugin.tvtpを削除してください。
Tvmaidのフォルダを削除してください。
レジストリは使用していません。
