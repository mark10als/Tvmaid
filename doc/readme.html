Tvmaid

概要

Tvmaidは録画予約アプリです。
Webブラウザを使って番組表を表示して録画予約し、TVTestと連携して録画します。

動作環境

OS

WindowsXP SP3以降。

TVTest

ver.0.7.16以降の32bit版。
64bit版では動きません。

対応ブラウザ

Firefox、Google Chromeの新しいバージョン。
IEでは動きません。

インストールとアンインストール

インストールマニュアルを参照してください。
操作方法はマニュアルを参照してください。
